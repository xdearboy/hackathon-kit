**_Разработка объектно_** - **_ориентированных классов для представления книг
и электронных книг, реализация их функциональности_**

**Создать модуль** book_module.py **, содержащий:**

```
 класс Book с указанными свойствами и методами
 класс EBook, наследующий Book, с дополнительными свойствами и
методами
```
**Создать основной скрипт** main.py **, демонстрирующий использование
классов:**

```
 создание объектов
 изменение данных
 аренда и возврат книг
 настройка электронной книги
 отправка ссылки по email (имитация)
```
**Требования к реализации**

1. Структура и файлы

Создать папку library

Внутри папки library разместить файл book_module.py

В корне проекта — файл main.py

2. Модуль book_module.py

Класс Book

Свойства (приватные):
_title — название книги (строка)
_author — автор (строка)
_year — год издания (целое число)
_is_borrowed — статус взятия (булево)
_borrow_date — дата взятия (datetime или None)
_borrow_period_days — срок займа (по умолчанию 14 дней)

Методы:
get_info() — возвращает строку с полной информацией
set_title(new_title) — изменить название
get_title() — получить название
set_author(author) — изменить автора
get_author() — получить автора
set_year(year) — изменить год
get_year() — получить год


borrow_book() — взять книгу взаймы (если свободна)
return_book() — вернуть книгу
is_borrowed() — проверить статус (возврат True/False)
extend_borrow_period(days) — продлить срок займа
change_author(new_author) — изменить автора
get_age(current_year) — узнать возраст книги

Класс EBook (наследует Book)

Дополнительные свойства:
_download_link — ссылка для скачивания
_format — формат файла

Методы:
set_format(new_format)
get_format()
set_download_link(link)
get_download_link()
send_download_link(email) — имитация отправки
update_download_link(new_link)
change_format(new_format)

3. Основной скрипт main.py

Импортировать классы из модуля
Создавать объекты Book и EBook
Производить операции:
изменение названия, автора, года
брать книгу взаймы и возвращать
устанавливать формат и ссылку для электронной книги
отправлять ссылку по email (вывод сообщения)

Обеспечить читаемость и структурированность кода
