# Cheatsheets

Короткий список шпор для быстрого поиска.

## Разделы

- `algorithms.md` — бинарный поиск, сортировки, решето, НОД/НОК, два указателя
- `classes.md` — OOP, dunder-методы, dataclass, наследование
- `collections.md` — `list`, `dict`, `set`, `tuple` и сложность
- `functions.md` — функции, аргументы, `*args`, `**kwargs`, рекурсия, лямбды
- `io.md` — ввод, вывод и сбор ответа
- `recursion.md` — рекурсия и DFS-шаблоны
- `stdlib.md` — `deque`, `Counter`, `heapq`, `bisect`, `itertools`, `lru_cache`
- `strings.md` — срезы, методы, `f-строки`, регулярки

## Как пользоваться

Открывай только нужный файл и бери оттуда шаблон кода, а не весь учебник сразу.
